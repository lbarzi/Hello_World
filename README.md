# Hello World Challenge

Raylytic challenge which aims to run a "Hello, World!" web page in Linux

## Getting Started

Just a simple set of shell script to get things done (almot) quickly and easily.

### Prerequisites

The task that defines the challenge is create a simple Web page, following some prerequisites:

* Debian-like distro;
* Simple webpage;
* Vagrant;
* Ansible;
* Documentation;
* Make the code available;
* Introduce the solution;


### Installing

#### Get the code

$ git clone git@github.com:lbarzi/Hello_World.git


#### Dive into the directory

$ cd Hello_World/

#### Fire up vagrant

$ vagrant up

And that's it, the "Hello World" will run after the bootstrap.
